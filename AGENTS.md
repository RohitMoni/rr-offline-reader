# RR Offline Reader

This app is intended to make Royal Road novels easy to download ahead of time and read later in a fully offline environment, such as on a plane. It is a small Preact/Vite PWA that stores novel metadata, downloaded chapters, and reading progress locally in IndexedDB, with a reader-focused UI for resuming, navigating chapters, and adjusting font size without relying on network access after content has been downloaded.

When changing the app, prioritize offline reliability, clear download/completion status, storage durability, and low-friction mobile reading UX.
